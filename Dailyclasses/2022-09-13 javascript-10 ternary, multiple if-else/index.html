Ternary Operator

Normal if-else

       if(condition){
         on true condition
        }else{
          on false of condition
        }

Ternary Operator

condition ? true statement : false statement

Multiple if-else chaining

      if(condition){
            on true condition 1
       }else if(condition 2){
            on true of  condition 2
       } else if(condition 3){
            on true of  condition 3
       } else if( condition 4){
             on true of  condition 2
       }else{
            when all conditions are failed/false
       }
